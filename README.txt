This is Pangaea Fusion, a lightweight, synchronized scouting infosystem. Pangaea Fusion is (c) 2019 Joseph Azrak @ Pangaea 6813

*** Dependencies ***
This software requires PHP >=6.0 and the MongoClient driver activated.
MongoClient may be downloaded here: https://s3.amazonaws.com/drivers.mongodb.org/php/index.html. A version is saved in critical_deps for convenience.

THIS SOFTWARE MUST BE RAN IN APACHE SERVER !! .htaccess rules are an important security measure here -- another webserver will ignore them,
and all visitors will be able to access database data etc. *Please use Apache Server.*
